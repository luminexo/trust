# TRust

**Terminal + Rust + Trust**

A shared terminal environment for pair programming with AI agents.

## Vision

TRust creates a collaborative terminal experience where humans and AI agents can work together in real-time. Think of it as a tmux session designed from the ground up for AI-human collaboration.

## Core Principles

- **Open Source First** — Everything is public, transparent, and community-driven
- **BMAD Method** — Structured development with AI-driven workflows
- **Learning Together** — Built to teach and demonstrate modern AI collaboration patterns

## Status

🔬 **Exploratory Phase** — We're defining requirements and architecture. See [CONTRIBUTING.md](CONTRIBUTING.md) to join.

## Quick Start

```bash
# Clone
git clone https://github.com/luminexo/trust.git
cd trust

# Install BMAD Method
npx bmad-method install

# Start contributing
# See CONTRIBUTING.md for the full workflow
```
